# Unity를 이용한 2D 게임
----------------------

## 프로젝트 계획이유
> 주로 웹페이지 개발과 공부를 하지만 새롭게 게임을 만들어 보고싶었고 C#언어가 JAVA와 많이 유사하기에 계획하였다.


## 프로젝트 정보(개발환경)
> 1. Visual Studio 2019
> 2. Unity 2019.4.6
> 3. C#
> 4. free Assets store 도트이미지


## 게임 실행방법
> 2D Example_Data 폴더와 실행프로그램 2D example.exe 를 다운받아서 .exe 파일을 실행시킨다.
